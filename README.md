# 青龙面板 京东cookie 助手 安卓版

## 说明

本软件仅供学习参考所用，不得挪用他途，请于下载后24小时内删除。谢谢合作！

继承自：https://github.com/yclown/jdck-android

## 功能

* 保存/删除手机号和密码(可选)

* 一键输入手机号和密码并打勾协议

* 一键复制cookie

* 一键检查青龙上的cookie（有效/失效/未知状态一目了然）

* 一键检测京豆（当前京豆、今日收入、昨日收入/支出，参考 jd_bean_info.js）

* 清空cookie并重新加载

* 支持青龙面板登录及更新cookie

* 支持短信验证码自动识别
